# PaperP

> [!IMPORTANT]
> PaperP is a Python reimplementation of [Paper](https://github.com/langningchen/paper).

> [!WARNING]
> This extension is a work in progress and may have bugs, incomplete features and break changes without notice.

This is **PaperP** - **P**en **A**DB **P**assword **E**asily **R**eset by **P**ython

PaperP is a tool to help you reset your YDDictPen ADB password easily.
It is designed to be user-friendly and efficient, making the process as smooth as possible.
